# Linux Log Analysis & Incident Investigation

## Role
Junior SOC / Blue Team Analyst

## Objective
Analyze Linux authentication logs to detect suspicious activity and demonstrate SOC-style incident reasoning.

## Environment
- OS: Ubuntu 24.04 LTS
- Platform: VMware
- Logs: auth.log, syslog
- Tools: grep, journalctl, awk, sed

## Scope
- Authentication events
- Failed login attempts
- Privilege escalation (sudo)
- Timeline reconstruction

## Outcome
- Identify abnormal patterns
- Assess risk
- Recommend defensive actions
